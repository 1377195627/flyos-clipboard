# flyos-clipboard

剪切板

## Build Dependencies (编译依赖)

* cmake
* qt5-tools

## Dependencies (依赖)

* qt5-base
* qt5-svg
* dtkwidget
* dtkcore

## Installation (安装)

* mkdir build
* cd build
* cmake -DCMAKE_INSTALL_PREFIX=/usr ..
* make
* sudo make install

## Usage (使用)

* flyos-cliboard 或者 点击启动器上的标题栏图标
